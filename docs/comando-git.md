# Comandos GitHub

# Inicializar el repositorio local
```

git init
```
- Referencia del repositorio local con el repositorio remoto
````
git remote add origin 
https://github.com/MIMI07-jk/Kirena01.git
````
# Validar referencia
````
dit remote -v
````

```
git add .
git commit -m "kirena1"
```

```
npm run start:dev
```
